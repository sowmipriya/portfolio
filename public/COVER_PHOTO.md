# Cover Photo Instructions

## Adding Your Cover Photo

1. **Save your cover photo** to the `public` folder in this project
2. **Name it exactly**: `cover-photo.jpg`
3. **Recommended format**: 
   - Wide/landscape image (16:9 or 21:9 aspect ratio works best)
   - At least 1920x1080 pixels for best quality
   - JPG, PNG, or WebP format
   - High quality image

## File Location

```
portfolio/
└── public/
    └── cover-photo.jpg  ← Place your cover photo here
```

## Supported Formats

- `.jpg` / `.jpeg` (recommended)
- `.png`
- `.webp`

## After Adding the Photo

1. The photo will automatically appear at the top of the Hero section
2. If the photo doesn't load, a beautiful gradient placeholder will be shown
3. Refresh your browser if you're running the dev server

## Tips

- Use a landscape/wide image for best results
- Make sure the file name is exactly `cover-photo.jpg` (case-sensitive)
- Keep file size reasonable (under 5MB recommended for fast loading)
- The cover photo displays at the top of the page, above your profile photo
